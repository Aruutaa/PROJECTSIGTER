# Design System

## Arah visual

Website menggabungkan nuansa immersive creative-tech dan portal smart city yang mudah dibaca. Visual dibuat catchy, tetapi struktur informasi tetap jelas.

## Palette

- Background: `#08111F`
- Mint: `#5EEAD4`
- Lavender: `#8B7CFF`
- Soft Pink: `#FF8CC6`
- Peach: `#FFD166`
- Sky Blue: `#6ECBFF`
- Red stress: `#FF6B6B`

## Komponen

- Glass card
- Rounded pill button
- Gradient text
- Animated ambient canvas
- Custom cursor
- WebGIS marker icon
- Badge kategori
- Dashboard metric card
- Timeline item

## Prinsip UI

1. Pengguna baru harus langsung tahu tombol utama: buka peta, lihat dashboard, cari safe zone.
2. Warna digunakan untuk membedakan kategori layer.
3. Mental health vibe tidak dibuat terlalu pucat; tetap colorful agar mudah diingat.
4. Elemen interaktif tidak boleh mengganggu keterbacaan.
5. Halaman WebGIS menjadi pusat sistem.
